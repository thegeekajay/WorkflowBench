Built for Production Teams

Everything you need to benchmark AI workflows without building a research platform.

YAML Case Format

Structured scenarios with context, inputs, expected outcomes, escalation rules, and forbidden actions. Easy to write, review, and version.

Deterministic Scoring

Four scoring dimensions — completion, escalation correctness, forbidden action violations, and required actions — with clear pass/fail criteria.

Provider Agnostic

Built-in adapters for OpenAI and Anthropic. Simple interface to plug in any model, agent, or custom workflow engine.

Failure Clusters

Reports don't just show a score — they group failures by type so you can see systemic issues at a glance.

Comparison Mode

Diff two runs to detect regressions and improvements. Know exactly which cases changed before you ship.

Local-First

No hosted service required. Run benchmarks from your terminal, store results in your repo, own your data.

Get Started in 60 Seconds

Install, run, and get a report — no configuration needed.

# Install
pip install -e ".[dev]"

# Run with the built-in echo adapter (no API key needed)
workflowbench run cases/ --adapter echo

# Or use OpenAI / Anthropic
workflowbench run cases/ --adapter openai --model gpt-4o

# Compare two runs
workflowbench compare reports/run_A.json reports/run_B.json

Scoring System

Transparent, deterministic, and weighted for enterprise workflows.
